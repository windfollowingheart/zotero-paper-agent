## 🚀 主要特性
Zotero Paper Agent 的特性包括：
- 🔗 **与 Zotero 集成**：此插件与 Zotero 高度集成，只需点击侧边栏图标。
- 💬 **易于使用**：通过输入框直接提问，类似于现在的 LLM 应用程序。
- 📄 **支持文件上传**：您可以上传文件并询问有关它们的问题。
